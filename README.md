# 情報エレクトロニクスファンド 月報分析ダッシュボード

## 概要

野村アセットマネジメント「情報エレクトロニクスファンド」の月報PDFを分析し、ファンドマネージャーの運用方針・銘柄選定の傾向を可視化するリサーチ支援ツールです。

> ⚠️ **重要**: 本ツールは投資助言ではありません。予測結果は過去データに基づく仮説であり、将来の運用成果を保証するものではありません。

## 主な機能

- 📊 **テーマ強度の時系列分析** — 半導体、AI、車載等のテーマの推移を可視化
- 🏢 **組入銘柄の変化追跡** — 上位10銘柄の入替・順位変動を月次で表示
- 📝 **運用方針テキスト分析** — キーワード抽出・売買シグナル検出
- 🎯 **新規組入候補の推定** — 説明可能な6要素スコアリングで候補を仮説提示
- 🧪 **バックテスト評価** — ローリング方式での予測精度検証
- 🔍 **データ品質レポート** — 抽出成否・欠損フィールドの可視化

## セットアップ

### 必要環境

- Python 3.9+
- モダンブラウザ（Chrome, Firefox, Edge）

### インストール

```powershell
cd "投資銘柄選定"
pip install -r requirements.txt
```

## 使い方

### 1. デモデータで試す（PDFなし）

```powershell
# サンプルデータ生成（36ヶ月分）
python src/generate_sample.py

# 分析実行
python src/analyze.py

# ダッシュボード起動
python -m http.server 8080
# → http://localhost:8080 をブラウザで開く
```

### 2. 実際のPDFで分析

1. `reports/` フォルダに月報PDFを配置
   - ファイル名例: `202301.pdf`, `2023-01_月報.pdf`, `2023年1月.pdf`
2. PDF抽出を実行:
   ```powershell
   python src/extract_reports.py
   ```
3. 分析実行:
   ```powershell
   python src/analyze.py
   ```
4. ダッシュボードを開く:
   ```powershell
   python -m http.server 8080
   ```

## 候補推定ロジック

6要素の重み付きスコアで候補を推定します:

| 要素 | 重み | 説明 |
|------|------|------|
| テーマ一致度 | 30% | 直近運用方針のテーマとの整合 |
| 過去採用頻度 | 20% | 過去に上位に登場した頻度 |
| 解説文類似度 | 15% | 運用方針テキストとの特徴類似 |
| 業種トレンド | 15% | 業種配分の変化方向との整合 |
| シグナル整合 | 10% | 売買示唆表現との整合 |
| 入替サイクル | 10% | 過去の入替パターンとの整合 |

## データ手動修正

抽出精度に問題がある場合、`data/reports_data.json` を直接編集し、`python src/analyze.py` を再実行できます。

## 今後の改善案

1. **精度改善**: SudachiPy/fugashi による形態素解析導入
2. **埋め込み類似度**: sentence-transformers によるテキスト類似度算出
3. **外部データ連携**: 株価データ、決算情報、アナリストレポートとの統合
4. **銘柄ユニバース拡張**: CSV読込による動的ユニバース設定
5. **シナリオ比較**: AI強気/設備投資鈍化等のシナリオ別推定
6. **ルールベース vs 類似度ベース**: モデル比較切替機能
